# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| Latest  | Yes       |

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it responsibly.

**Email:** security@mountainash.io

Please include:

- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if any)

We will acknowledge receipt within 48 hours and aim to provide a resolution timeline within 5 business days.

## Disclosure Policy

- Please do not open public issues for security vulnerabilities.
- We will coordinate disclosure timing with you after a fix is available.
